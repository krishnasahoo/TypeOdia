TypeOdia

This is a tweaked jquery.ime project to enable users run it offline with out server-application. It's is also customized for Odia (a.k.a. Oriya) language. It only supports Mozilla Firefox web browser by default. With a little tweaking in browser configuration can be used on any web browser.
